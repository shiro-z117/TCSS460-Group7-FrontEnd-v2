Movies API
 1.0.0 
OAS 3.0
API for movies from the last 30 years. Provides routes for listing, retrieving, creating, updating, deleting, and analyzing movies. Supports filtering, pagination, and statistics grouped by year, genre, or other dimensions.

Servers

https://dataset-web-api.onrender.com/api - Render (production)
